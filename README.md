# 🔢 Número Secreto

Projeto desenvolvido com **HTML**, **CSS** e **JavaScript**.

## 🎮 Sobre

O objetivo do jogo é adivinhar o número secreto gerado pelo sistema. A cada tentativa, o jogo informa se o número é maior ou menor até você acertar.

## 🚀 Tecnologias

- HTML
- CSS
- JavaScript

## ▶️ Como executar

1. Baixe ou clone o projeto.
2. Abra o arquivo `index.html` no navegador.
